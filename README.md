#  Blinkit Dashboard

A clean and interactive data dashboard built to analyze and visualize Blinkit (grocery delivery) business performance. This project helps in understanding sales trends, product performance, customer behavior, and outlet insights using visual analytics.

---

##  Project Overview

The **Blinkit Dashboard** is a data visualization project designed to transform raw retail data into meaningful insights. It presents key business metrics in a simple and easy-to-understand format for better decision-making.

This dashboard focuses on:

* Sales performance
* Product category insights
* Outlet analysis
* Customer trends
* Revenue patterns

---

##  Objectives

* Analyze total sales and revenue trends
* Understand product category performance
* Study outlet location and size impact
* Identify high-performing items
* Present insights through interactive visuals
